
# TCP-File-Operations #
 Network Programming using TCP.

#### Server ####
        pwd
        ls   [pathname]
        cd    pathname
        mkdir pathname
        rmdir pathname       
        rm    pathname       (rm pathname)
        get   pathname       (cp pathname to client side)
        put   pathname       (cp pathanme to server side)

#### Local Client ####
        lpwd
        lls   [pathname]
        lcd    pathname
        lmkdir pathname
        lrmdir pathname       
        lrm    pathname       (rm pathname)
        get   pathname       (cp pathname to client side)
        put   pathname       (cp pathanme to server side)
        quit                 (Client exits)
